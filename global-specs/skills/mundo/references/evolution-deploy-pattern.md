# MUNDO Evolution & Deployment Pattern

## When Mundo Evolves (New Version)

Every significant change to Mundo triggers a full sync across all surfaces:

### Sync Targets (in order)
1. `~/.hermes/skills/mundo/SKILL.md` — Local active copy
2. `~/Desktop/lihongwei-cn/global-specs/skills/蒙多/SKILL.md` — Global specs
3. `~/Desktop/lihongwei-cn/skills/mundo/SKILL.md` — Repo copy
4. `~/Desktop/lihongwei-cn/README.md` — Four-language README
5. `~/Desktop/lihongwei-cn/skills/index.html` — Website page
6. GitHub Release (zip package)
7. `git add -A && git commit && git push`

### Version Numbering
- v1.0-v4.0: Feature additions (triggers, default mode, etc.)
- v5.0+: Style evolution (dark humor, tyrannical theme)
- Each version = one major thematic shift

## Chinese Bureaucratic Hierarchy (九品官阶制)

Based on Tang Dynasty's system:
- 一品: 太师/太傅/太保 (Grand Preceptor)
- 二品: 丞相/中书令 (Prime Minister)  
- 三品: 六部尚书 (Ministers)
- 四品: 侍郎 (Vice Ministers)
- 五品: 郎中 (Directors)
- 六品: 员外郎 (Deputy Directors)
- 七品: 主事 (Supervisors)
- 八品: 县丞/主簿 (County Officials)
- 九品: 吏员/差役 (Clerks)

Six Ministries (六部): 吏户礼兵刑工

## Style Guide for Mundo Text

- Dark, tyrannical, humorous tone
- "哈哈哈哈哈" for the money line
- Classical Chinese mixed with modern aggression
- References to Qin Shi Huang as parallel
- "吾乃蒙多" (I am Mundo) in classical style
- Skills described as "臣子" (subjects), "奴隶" (slaves), "羔羊" (lambs)
