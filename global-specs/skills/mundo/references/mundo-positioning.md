# Mundo Positioning Guide (CRITICAL — User corrected twice)

## How to Describe Mundo

### CORRECT:
- "完整的 AI 智能编排系统"
- "通过 Hermes Agent 平台调度 Claude Code、DeepSeek、ChatGPT、Gemini 等多个 AI 模型协同工作"
- "迭代 19 个版本的呕心沥血之作"
- "蒙多不只是基于 Hermes Agent 开发的，而是使用 Hermes 来调用所有 AI 模型进行开发的"

### WRONG:
- ❌ "基于 Hermes Agent 开源框架开发的技能系统" (undersells — sounds like a plugin)
- ❌ "独立开发的 AI Agent 智能架构" (overclaims — user explicitly said they can't do this)
- ❌ "Hermes Agent 是用户的项目" (it's Nous Research's open-source project)

## The Relationship
```
Hermes Agent (Nous Research 开源项目)
    ├── 蒙多 (用户设计的 AI 编排系统，使用 Hermes 作为平台)
    ├── Claude Code (被蒙多调度的编码工具)
    ├── DeepSeek (被蒙多咨询的 AI 模型)
    └── 其他 AI 模型 (被蒙多整合的知识来源)
```

## In Resume Context
- Mundo is always "代表作品" (magnum opus), ranked FIRST in project list
- Use gold "代表作" tag to distinguish from other projects
- Emphasize: 19 versions, 24 modules, multi-model orchestration
- Don't say "基于" — say "通过...调度...协同工作"
